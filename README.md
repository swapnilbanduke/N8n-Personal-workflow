# Chat Assistant (Streamlit)

This repository contains a small Streamlit chat client.

Files included:
- `app.py` — main Streamlit application (keep this file)

Requirements
---------
- Python 3.8+
- Install dependencies:

```powershell
pip install streamlit requests
```

Run
----

```powershell
streamlit run app.py
```

Notes
-----
- The webhook endpoint used by the app is configured inside `app.py` as `DEFAULT_WEBHOOK_URL`.
- Use the Export button in the sidebar to download conversation history.
